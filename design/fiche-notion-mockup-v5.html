1

Le principe positionnel

12 et 21 utilisent les mêmes chiffres mais ne valent pas la même chose. La différence, c'est la position. Chaque rang vaut dix fois le rang à sa droite : c'est le système décimal.

3
centaines
3×10² = 300
4
dizaines
4×10¹ = 40
5
unités
5×10⁰ = 5

345 = 3×10² + 4×10¹ + 5×10⁰ = 300 + 40 + 5.

Dans 4 062, le zéro n'est pas « rien » : il occupe la position des centaines. Sans lui, 462 ≠ 4062.
2

L'indice de base

Le petit chiffre en bas à droite indique dans quelle base est écrit le nombre. Sans indice → base 10 par défaut.

345pas d'indice → base 10 · trois cent quarante-cinq
1101₂indice 2 → base 2 (binaire) · vaut 13 en décimal
1F₁₆indice 16 → base 16 (hexadécimal) · vaut 31
Piège classique. 1101₂ ≠ 1101. Repérer l'indice avant tout calcul, c'est le réflexe n°1.
3

Les autres bases

Le principe positionnel fonctionne avec n'importe quelle base. Dans une base b, on utilise b chiffres (0 à b−1), et on passe au rang suivant dès qu'on a compté b fois.

Convertir vers la base 10

D'une base b quelconque vers le décimal, en trois gestes.

Question typique : « Convertir 1101₂ en base 10. » Vous savez déjà le faire en base 10 : on applique exactement la même logique.
1
Numéroter les positions de droite à gauche, à partir de 0.
Nombre : 1 1 0 1 Position : 3 2 1 0 ← toujours 0 à droite
2
Même calcul qu'avec 345, mais la base est 2.
pos. 3 → 1 × 2³ = 8 pos. 2 → 1 × 2² = 4 pos. 1 → 0 × 2¹ = 0 pos. 0 → 1 × 2⁰ = 1
3
Additionner.
8 + 4 + 0 + 1 = 13 ✓ 1101₂ = 13 en décimal

L'indice de base

Pour chaque nombre, dire dans quelle base il est écrit :  a) 101₂  b) 1F₁₆  c) 347  d) 25₈

Voir la correction
  • a) 101₂ → base 2 (binaire)
  • b) 1F₁₆ → base 16, F = 15
  • c) 347 → base 10 par défaut
  • d) 25₈ → base 8 (octal)
Échauffement

Valeur positionnelle

Dans le nombre 4 253, quelle est la valeur des chiffres 4, 2, 5 et 3 ?

Voir la correction
  • 4 → 4×10³ = 4 000
  • 2 → 2×10² = 200
  • 5 → 5×10¹ = 50
  • 3 → 3×10⁰ = 3

Type concours · Exercice 7

Travailler dans une autre base

Dans une classe de CM2, un enseignant propose : « En base 5, on utilise les chiffres 0, 1, 2, 3, 4. Écrire les nombres de 0 à 12 en base 5, puis calculer 23₅ + 14₅. »

1. Écrire les nombres de 0 à 12 en base 5.
2. Calculer 23₅ + 14₅ en base 5, puis vérifier en base 10.
3. Quel est l'intérêt pédagogique pour des élèves de CM2 ?

Voir le corrigé complet

23₅ + 14₅ : unités 3+4 = 7 = 1×5+2 → écrire 2, retenir 1 ; dizaines 2+1+1 = 4 → 42₅. Vérification : 13+9 = 22 ; 42₅ = 22 ✓

Travailler en base 5 permet aux élèves de décontextualiser la numération décimale : ils comprennent que « 10 » signifie « une unité d'ordre supérieur », quelle que soit la base (Éduscol, guide numération).

Erreur type

Erreur réelle de candidat, analysée à partir des rapports de jury.

Copie d'un candidat : convertir 13 en base 2.

13 ÷ 2 = 6 reste 1 6 ÷ 2 = 3 reste 0 3 ÷ 2 = 1 reste 1 1 ÷ 2 = 0 reste 1 Résultat : 1011₂

a) Cette réponse est-elle correcte ?  b) Identifier l'erreur.  c) Quelle vérification l'évitait ?

Voir le corrigé

Erreur : sens de lecture des restes inversé.

  • Restes lus de haut en bas (1,0,1,1) au lieu de bas en haut (1,1,0,1).
  • Réponse correcte : 1101₂. Vérification : 8+4+0+1 = 13 ✓

Quelle est la différence entre un système additif et un système positionnel ?

Cliquer pour révéler
Réponse

Additif : valeur fixe par symbole (romain : VII = 7). Positionnel : la valeur dépend de la position (le 3 dans 305 vaut 300, dans 53 vaut 3).
